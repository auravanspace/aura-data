# aura-data
我曾经做过的数据分析外包单子

## 叠甲
鉴于国内数据分析市场的奇怪供需状态，本人已无力吐槽。肉眼可见的培训班割韭菜加上普遍落后时代的数据分析能力，需求和能力都让人无语的甲方。为了避免数据分析这个有趣的里世界彻底崩坏，本人把过去做过的数据分析接单内容都开源在这边。希望各位能一瞥数据分析的乐趣。
注：一下内容源于真实市场需求。同样，ai一直在过程中发挥作用，利用好ai。他们是摸鱼的好帮手。

## 准则
不流动的信息毫无价值。
完成自己擅长的部分。不要总是尝试在别人的专业内超过别人。
